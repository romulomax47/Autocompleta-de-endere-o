## AUTO COMPLETA DE ENDEREÇO

### OBJETIVO
Ao preenche o campo 'Digite seu Cep' o solfware buscará seu endereço e preenchera automaticamente os campos Rua, Cidade, Estado.


### Ferramentas utilizadas 🏗️.
- Html5
- Css
- Javascript
- Bootstrap
